Statistical Analysis Calculator

Shows the mean, median, mode, variance, sum, 
standard deviation, average deviation, coefficient
of deviation, total, square total, and variance for
an array of values using either the population or
sample method. The code includes a function
to sort an array and a digital display. Has
unlimited undo feature and selectable listbox
sorting. Code comments are included. Update:
removes scientific notation so that results are 
decipherable, without losing any accuracy. Also
added rounding from 1 to 10 decimal places.
